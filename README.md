# React Design System (app)

[Demo](https://adrienloup.github.io/ds/)

## Requirements

- `node` : `^18.18.2`
- `npm` : `^9.8.1`

## Installation

```bash
nvm use
npm install
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
```

```bash
npm run preview
```

## Code Quality Check

```bash
npm run lint
npm run lint:fix
```
